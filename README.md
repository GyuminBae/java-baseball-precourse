# java-baseball-precourse
## 구현해야할 기능 목록
- 1부터 9로 이루어진 서로 다른 3개의 수를 생성
- 사용자로부터 3개의 수를 입력받음
- 사용자가 입력한 수와 생성한 수를 비교하여 볼과 스트라이크를 판정
  - 같은 수가 같은 자리에 있으면 스트라이크
  - 같은 수가 다른 자리에 있으면 볼
  - 같은 수가 전혀 없으면 낫싱
- 3스트라이크일 경우 게임 종료
- 게임 종료 후 다시 시작하거나 완전히 종료할 수 있음
- 플레이어가 잘못된 값을 입력하면 IllegalArgumentException throw 후 종료
